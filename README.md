\# Streamlit Uber Pickups Project



A beginner data science project built using Python and Streamlit.



This project analyzes Uber pickup data in New York City and visualizes it using interactive charts and maps.



\## Features



\- Load and cache large datasets using `st.cache\_data`

\- Display raw dataset

\- Interactive histogram of pickups by hour

\- Filter pickups using slider

\- Visualize pickup locations on a map

\- Beginner-friendly Streamlit application



\## Technologies Used



\- Python

\- Streamlit

\- Pandas

\- NumPy

\## Installation



Clone the repository:



```bash

git clone https://github.com/aarju-24/Streamlit\_learning\_project.git

```



Move into the project folder:



```bash

cd Streamlit\_learning\_project

```



Install dependencies:



```bash

pip install -r requirements.txt

```



Run the Streamlit app:



```bash

streamlit run uber\_pickups.py

```
